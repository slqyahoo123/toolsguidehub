# 文章布局快速参考

## 布局结构

```
<article>
  <header>          ← 禁止广告
    H1标题
    介绍摘要
    元数据
    特色图片
  </header>
  
  [AD ZONE 1]      ← 可选：顶部广告
  
  <main>            ← 禁止广告（正文）
    <section>
      文章内容
    </section>
    
    [AD ZONE 2]    ← 可选：文章内广告（长文章）
  </main>
  
  <nav>             ← 禁止广告
    相关文章链接
  </nav>
  
  [AD ZONE 3]      ← 可选：底部广告
  
  <footer>          ← 禁止广告
    底部信息
  </footer>
</article>
```

## 标题层次

```
H1 (唯一) - 文章标题
  └─ H2 - 主要章节（6-8个）
      └─ H3 - 子章节（3-4个 per H2）
```

## 广告位置

### ✅ 允许的广告位置

1. **AD ZONE 1**: 标题后，正文前
   - 要求: 文章 ≥ 500字
   - 类型: 横幅广告

2. **AD ZONE 2**: 正文中间
   - 要求: 文章 ≥ 1000字
   - 位置: 至少300字后
   - 类型: 矩形广告

3. **AD ZONE 3**: 正文后，相关文章前
   - 要求: 所有文章
   - 类型: 横幅广告

### ❌ 禁止广告的区域

- ❌ `<header>` - 标题区域
- ❌ `<main>` 正文段落内
- ❌ `<nav>` - 导航区域
- ❌ `<footer>` - 页脚区域

## AdSense政策要点

### 必须遵守

- ✅ 内容为主（80%+）
- ✅ 广告为辅（<20%）
- ✅ 不遮挡内容
- ✅ 每页至少500字
- ✅ 移动端友好

### 禁止行为

- ❌ 广告在标题区域
- ❌ 广告嵌入段落
- ❌ 广告遮挡内容
- ❌ 广告在导航区域
- ❌ 弹出式广告
