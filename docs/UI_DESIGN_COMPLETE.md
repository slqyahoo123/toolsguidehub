## 🎨 设计理念

基于网站特色和内容，我设计了一个**专业、现代、易读**的UI系统，专注于：

1. **内容优先** - 设计服务于内容，不喧宾夺主

3. **易读易用** - 优化阅读体验和导航

## 🎨 设计系统

### 颜色系统
- **主色**：`#0c7cd6` (primary-600) - 专业蓝色
- **强调色**：`#14b8a6` (accent-500) - 青绿色
- **文字**：`gray-900` (标题), `gray-800` (正文), `gray-700` (次要)

### 字体系统
- **H1**：4xl-7xl (40px-72px)
- **H2**：2xl-3xl (24px-32px)
- **H3**：xl-2xl (20px-24px)
- **正文**：base-lg (16px-18px)
- **行高**：1.7 (正文), 1.2-1.4 (标题)

### 间距系统
- **页面间距**：py-16 md:py-20 (64px-80px)
- **章节间距**：mb-12-14 (48px-56px)
- **卡片间距**：gap-6 (24px)

### 组件设计
- **按钮**：主要按钮（primary-600背景），次要按钮（白色+边框）
- **卡片**：2px边框，hover时阴影增强
- **图标**：统一的SVG图标，渐变背景

## 🎯 设计亮点

1. **品牌一致性**：Logo图标贯穿整个网站
2. **视觉层次**：清晰的标题、正文、次要文字层次
3. **交互反馈**：所有可交互元素都有清晰的hover效果
4. **专业外观**：渐变、阴影、圆角的现代设计
5. **内容优先**：设计不喧宾夺主，专注于内容展示
