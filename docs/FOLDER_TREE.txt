关键路径说明
============

路由映射：
----------
/                           → app/page.tsx (首页)
/[category]                 → app/[category]/page.tsx (分类列表)
/[category]/[slug]          → app/[category]/[slug]/page.tsx (文章详情)
/about                      → app/(marketing)/about/page.tsx
/contact                    → app/(marketing)/contact/page.tsx


内容映射：
----------
content/articles/[category]/[slug].md
    ↓
lib/content/loader.ts (加载和解析)
    ↓
app/[category]/[slug]/page.tsx (渲染)


URL示例：
--------
/web-development/nextjs-guide
    ↓
content/articles/web-development/nextjs-guide.md
    ↓
app/web-development/nextjs-guide (SSG生成)


扩展性设计
==========

支持500+文章的策略：
-------------------
1. 按分类组织，每个分类文件夹管理相关文章
2. 使用 generateStaticParams 分批生成
3. 利用 Next.js 的增量静态生成 (ISR)
4. CDN 缓存所有静态页面

添加新分类：
-----------
1. 在 content/articles/ 创建新文件夹
2. 在 config/categories.ts 添加配置
3. 自动支持路由 /[new-category]

添加新文章：
-----------
1. 在对应分类文件夹创建 .md 文件
2. 运行构建，自动生成页面
3. Sitemap 自动更新
